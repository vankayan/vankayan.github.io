Course Description

This course offers an in-depth exploration of fundamental web application development, covering both front-end technologies like HTML5, JavaScript, and CSS, as well as back-end tools such as PHP and MySQL. Through hands-on exercises, students will delve into web application vulnerabilities and attacks, gaining practical experience in identifying and mitigating security risks. Emphasis is placed on introducing secure programming principles and best practices to safeguard against potential vulnerabilities and attacks in web applications.

Course Outcomes

Lab-1

  • Explored packet analysis techniques using Wireshark.
  • Introduced to telnet and curl utilities.
  • Examined the process of packet transmission and response generation in HTTP requests.
  • Experimented with C, HTML, PHP, and echo commands to grasp the functionality of packets and the workings of HTTP streams.

Lab-2

  • Acquired skills in integrating HTML with JavaScript.
  • Implemented CSS styling to enhance the appearance of HTML pages.
  • Utilized Ajax to send both GET and POST requests.
  • Programmed to listen for HTTP requests and dynamically display the content within designated elements.

Hackathon-1

  • Explored the concept of cross-site scripting (XSS) attacks during the hackathon.
  • Successfully completed levels 0 to 6, involving the analysis of source code.
  • Received helpful hints that facilitated progress through the levels.
  • Gained insight into the execution of XSS attacks on websites.
  • Acquired strategies for mitigating and preventing XSS attacks.
  • The hackathon consisted of two primary tasks, each comprising multiple sub-tasks, with grades allocated accordingly.
